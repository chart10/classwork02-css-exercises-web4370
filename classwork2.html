Exercise 6

For this website to work, every department needs to be able to update the content of their page without disrupting the design of the site. Consistency is key here. Have an exernal CSS file that each department's HTML page can link to is essential.

Each department can use several basic HTML tags to organize their page, such as h1 through h6 and p, but it will also be useful to designate certain elements with a class. For example class="heading" could be used to group together all of the page heading information. The styling could be decided on the external CSS file, and the content for each page could be decided by each department, without changing the coherence of the wesite.

Some examples of tags and classes that could be used:

  • h1 through h6 tags
  • p (paragraph) tag
  • article tag
  • body tag
  • table tag
  • a (hyperlink) tag
  • heading class
  • footer class
  • navigation-bar class
  • exam class
  • instructor class

In the CSS file we can edit various properties for a category of tag or class and it will effect all exaples of that tag/class throughout the whole site. Everything from text color, background-color, font-family, box-shadow or the size of the various box model properties can be managed from this one file and it will change all of the linked HTML pages in the same way at the same time.
